# Bedrock 3D 引擎 (Bedrock 3D Integration)

PeroCore 使用了 YSM 模组同款的 Minecraft Bedrock 格式 3D 模型，作为桌面宠物的核心视觉表现。这种格式不仅文件体积小巧，而且支持复杂的骨骼动画和动态换装，非常适合作为常驻桌面的 AI 伙伴。

## 🕹️ 桌面端操作指南 (Desktop Usage)

在 PeroCore 桌面端 中，Pet3DView 是用户与 AI 交互的主要窗口。

### 1. UI 组件概览

当你的鼠标悬停在模型身上时，会激活 UI 覆盖层 (UI Overlay)。

*   **状态标签 (Status Tags)** (左上角):
    *   `❤️ Mood`: 当前情绪状态（如：开心、生气、疲惫）。
    *   `✨ Vibe`: 当前环境氛围（如：轻松、紧张）。
    *   `💭 Mind`: 当前的心理活动（如：“主人正在干嘛呀？”）。
*   **悬浮触发器 (Floating Trigger)**: 一个跟随 UI 缩放的动态光球，用于唤起输入框。
*   **输入框 (Chat Input)**: 按 `Enter` 发送消息。在工作模式下会被禁用。
*   **工具栏 (Toolbar)**:
    *   `🎨 外观设置`: 快速更换衣服、配饰。
    *   `🔄 重载`: 重新加载模型资源（当模型出现显示异常时使用）。
    *   `📏 调整大小`: 切换窗口尺寸模式。
    *   `🎙️ 语音模式`: 切换 **关闭 / VAD (自动感应) / PTT (按住说话)**。
    *   `💬 聊天`: 打开独立的聊天窗口。
    *   `⚙️ 面板`: 打开设置与仪表盘。

### 2. 模型交互 (Model Interaction)

*   **视线追踪 (Head Tracking)**: 模型的头部和眼睛会自动跟随你的鼠标移动。
*   **抚摸 (Petting)**: 在头部区域快速移动鼠标，会触发抚摸反馈（如眯眼、爱心特效）。
*   **平移与旋转 (Pan & Rotate)**:
    *   在演示插件中：**左键拖拽**旋转视角，**Ctrl + 左键**平移模型。
    *   在桌面客户端：**左键拖拽**可以搬运窗口，实体区域可以触发交互，右键可以旋转模型。
*   **物理反馈**: 拖动时，模型会呈现出“被拎起来”的物理惯性效果（身体会根据拖动方向摆动）。

### 3. 对话与气泡 (Dialogue Bubbles)

AI 的回复会以气泡形式显示在头顶。

*   **思考中 (Thinking)**: 当 AI 正在推理时，气泡会显示“努力思考中...”，此时你可以点击气泡查看详细的**思维链 (Chain of Thought)**。
*   **动作演出 (Action)**: 如果回复中包含 `*开心*` 或 `*生气*` 等动作标记，模型会自动播放对应的动画。
*   **自动收起**: 长时间无交互后，气泡会自动收起以减少对屏幕的遮挡。

### 4. 语音交互 (Voice Interaction)

PeroCore 支持两种语音模式：

1.  **VAD (自动感应)**: 自动检测环境音量，当你说话时自动录音，说完自动发送。适合安静环境。
2.  **PTT (按住说话)**: 屏幕上会出现一个悬浮的麦克风图标（或使用快捷键 `Alt+Shift+V`），按住说话，松开发送。适合嘈杂环境。

## 🛠️ 技术实现细节

### 渲染管线 (Rendering Pipeline)

我们使用了 `Three.js` 进行底层渲染，并编写了专门的加载器来解析 Bedrock 格式。

*   **Geometry**: 解析 `.geo.json` 文件，构建层级化的 `THREE.Group` 和 `THREE.Mesh`。
*   **Texture**: 动态加载 `.png` 贴图，支持 UV 动画。
*   **Animation**: 使用 `AnimationManager` 解析 `.animation.json`，支持 `Molang` 表达式进行复杂的关键帧插值。

### 隐私与性能

*   **离屏渲染**: 3D 渲染在独立的 GPU 进程中进行，不会占用过多的 CPU 资源。
*   **事件穿透**: 默认情况下，透明区域允许鼠标穿透（点击到后方的窗口），只有实体部分会拦截点击。
