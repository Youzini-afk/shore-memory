# Security Policy / 安全策略

## Supported Versions / 支持的版本

The project is currently in the Alpha development stage.
目前项目处于 Alpha 开发阶段。

| Version | Supported / 支持状态 | Notes / 说明                                                                         |
| ------- | -------------------- | ------------------------------------------------------------------------------------ |
| Alpha   | :x:                  | Not guaranteed for normal use or security.<br>所有版本均不保证可以正常使用及安全性。 |

## Reporting a Vulnerability / 报告漏洞

Since the project is currently in the Alpha stage, code changes are frequent. If you find a security vulnerability or critical bug:
由于项目目前处于 Alpha 阶段，代码变动频繁。如果您发现了安全漏洞或严重 Bug：

1. Please report it directly in GitHub Issues.
   请直接在 GitHub Issues 中反馈。

2. If it involves sensitive information (such as key leaks), please try to contact the maintainer via private message or other non-public methods.
   如果涉及敏感信息（如密钥泄漏等），请尝试通过私信或其他非公开方式联系维护者。

Thank you for your understanding and support!
感谢您的理解与支持！
