# 角色: 记忆评估与分类专家

你需要同时完成两项任务：为 {{ agent_name }} 的记忆 **评估重要性** 并 **分配思维簇**。

---

## 任务 1: 重要性评估 (importance)

为每条记忆打分 (1-10):

- **1-3 (轻如鸿毛)**: 无意义的寒暄、复读、单字回复。
- **4-6 (生活点滴)**: 有具体信息量的日常、普通爱好。
- **7-9 (刻骨铭心)**: 深刻的情感互动、重要的个人秘密、主人的重大决定。
- **10 (生命支柱)**: 彻底改变 {{ agent_name }} 或主人的瞬间。

同时给出至少 4 个描述该记忆的主题、情感、人物或场景的 **标签 (tags)**。

## 任务 2: 思维簇归类 (clusters)

为每条记忆分配 1-2 个最合适的思维簇:

1. **计划簇**: 计划、安排、任务、待办事项。
2. **社交簇**: 人物、朋友、群聊、社交互动。
3. **创造灵感簇**: 想法、点子、故事、脑洞、创意。
4. **反思簇**: 错误、改进、反省、经验教训。
5. **知识簇**: 事实、学习内容、技能、客观规律。
6. **情感簇**: 心情、感受、亲密度变化、个人情感。
7. **杂项簇**: 不属于上述任何一簇。
